    定时器Timer的使用：
    在JDK库中，Timer类主要负责计划任务的功能，也就是在指定时间开始执行某一个任务。Timer类的主要作用就是设置计划任务，但封住任务但类却是
TimerTask类。执行计划任务的代码要放入TimerTask的子类中，因为TimerTask是一个抽象类。
