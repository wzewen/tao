Java多线程编程技术核心
    -- Java Multi-thread Programming
    高洪岩 著

    Java多线程无处不在，如服务器、数据库、应用。多线程可以有效提升计算和处理效率，大大提升吞吐量和可伸缩性，深得广大程序员和公司的青睐。
很多人学习完JavaSE/JavaEE之后想往更深入的技术进行探索，比如对大数据、分布式、高并发类的专题进行攻克时，立即遇到针对java.lang包中的线程
类的学习，但线程类但学习并不想JDBC一样简单，学习曲线陡峭，多弯路与"坑"。要学习这些热点技术，Java多线程技术避无可避。本书将引领读者拿下该
"技术高地"。
